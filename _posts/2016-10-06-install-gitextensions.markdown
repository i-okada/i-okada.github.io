WindowsにGitExtensions(version: 2.48.05)をインストールしたときのメモ。

## 手順

1. Required Software: ３つともチェック（MsysGit, Install KDiff, GitCred）
2. Select SSH Client: OpenSSH(Tera Termインストール済みなのでこちらを選択)
3. ユーザ名とアドレスを設定
4. finish
